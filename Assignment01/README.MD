# 第一次培训作业 - 多线程

## 背景

现有一考试系统需要为一批学生添加账号，账号名称是由学号生成的加密密码，现在已知这批学生的姓名及学号，需要生成账号及密码。

这些学生的姓名及学号信息包含在`input.txt`文件中，该文件大约包含两万多行，每行都是由“姓名 学号”的形式构成，以UTF8格式存储。

## 任务

现在需要为这些学生生成账户，账号即学号，密码可以使用`GenerateEncryptedPassword`函数生成，生成之后写入到一个csv文件中。

现在已经写好了一个版本，但该版本运行耗时过长，现在需要你使用多线程优化该程序。

## 提示

- 线程数量过多可能会导致总体性能下降
- 线程的创建开销大，因此不要一次性只做非常少的事请
- 你不需要像示例代码一样实时输出处理了多少行数据，只需要在完成所有处理后输出耗时即可
